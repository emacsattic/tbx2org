## Import Eastgate tinderbox files into org-mode

A simple tool to convert [tinderbox](http://www.eastgate.com/Tinderbox/) outlines into [org-mode](http://www.orgmode.org) format.

### Usage

- install the package:
```
    M-x package-install RET tbx2org RET
```

- run:

```
    M-x tbx-import-file
```
